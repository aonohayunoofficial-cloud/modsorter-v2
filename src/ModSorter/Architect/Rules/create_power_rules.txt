=== Create 動力ルール（第1層: 伝達） ===

# 大前提
- 動力は回転力であり、応力(su)という単位で表す。
- 1つの動力ライン全体で「発生する応力 >= 消費する応力」でなければ動かない。
- 動力の伝達ロスは無い。どれだけ遠くまで伝えても出力(応力)は同じ。
- 1本のラインに矛盾した向きの回転をぶつけると、ブロックが破壊されてドロップする。

# 軸(axis)の基本
- shaft は応力を直線的に伝達する。axis 方向に一直線で繋がる。
  - axis=x は東西、axis=y は上下、axis=z は南北。
  - 例: axis=z の shaft は north/south 方向の隣に置くと接続する。
- cogwheel も axis を持つ。同じ axis の shaft を軸方向の隣に置くと伝達する。
- 軸(axis)が揃っていない shaft 同士、shaft と cogwheel の接続では動力は伝わらない。

# 歯車の噛み合わせ
- cogwheel 同士は、軸に垂直な方向（正面から見て上下左右）に隣接させると噛み合う。
- 噛み合った歯車は互いに同じ回転数で、回転の向きが必ず逆転する。
  - 例: A が右回転なら、噛み合う B は左回転。
- 歯車は1個噛み合うごとに回転の向きが反転する。
  - 同じ向きが欲しい場合は、歯車を偶数個挟むか、shaft で軸を延長する。

# 増速・減速（large_cogwheel と cogwheel）
- large_cogwheel と cogwheel は、正面から見て斜め方向に隣接させて噛み合わせる。
- 比率は 1:2。回転の向きは逆転する。
  - 増速: large_cogwheel を伝達元、cogwheel を伝達先にすると、cogwheel は2倍速になる。
  - 減速: cogwheel を伝達元、large_cogwheel を伝達先にすると、減速する。
- large_cogwheel 同士は直角に噛み合わせ可能。回転の向きは逆になる。

# 向きの変更・制御
- gearbox: 水平方向に入出力2軸。回転数は変えず、入力と逆向きを出力する。
- vertical_gearbox(垂直ギアボックス): 水平と垂直に各1軸。入力と逆向きを出力。
- clutch(クラッチ): レッドストーン信号が入っている間、動力の伝達を止める。
- gearshift(ギアシフト): レッドストーン信号が入っている間、回転の向きを逆転させる。
- encased_chain_drive(ケース入りチェーンドライブ): 接続した同種ブロックに同じ向きの回転を与える。レンチで直角に曲げられる。

# 機械への動力入力
- 機械ブロック(millstone, mechanical_press 等)に動力を入れるには、
  その機械が要求する面に cogwheel か shaft を axis を合わせて隣接させる。

# 絶対にやってはいけないこと（破損する）
- 逆向きの回転（右回転と左回転）を同じ軸にぶつける。
  → 繋いだ瞬間に競合して shaft が壊れる。
- 速度(rpm)が食い違う回転を同じラインに合流させる。
  → 速度競合でラインが停止・破損する。

# 合流による出力増（推奨される構成）
- 回転の向きと速度が一致した動力源どうし（右回転と右回転、または左回転と左回転で、かつ同じrpm）
  であれば、複数を同じラインに合流させてよい。
- これは禁止ではなく推奨で、供給応力(Stress Capacity)が合算され、ライン全体の出力上限が増える。
  例: water_wheel を複数、向きと速度を揃えて1本のラインに束ねると、より多くの機械を駆動できる。
- 注意: 歯車は1個噛むごとに向きが反転するため、同じ動力源から分岐した回転でも、
  合流点までに通った歯車の枚数が片方だけ奇数・片方だけ偶数だと向きが逆になりぶつかって破損する。
  合流させる場合は、合流点で回転の向きと速度の両方を一致させること。

# 主な動力源（出力の目安、0.5.1基準）
- hand_crank(ハンドクランク): 256su / 32rpm
- water_wheel(水車): 128su / 8rpm（接する液体流の面数に関わらず一定）
- large_water_wheel(大きな水車): 512su / 4rpm（水車の1/2のrpm、4倍のsu）
- windmill_bearing(風車ベアリング): 帆8枚で512su/1rpm、8枚ごとに+512su+1rpm、最大128枚で8192su/16rpm
- steam_engine(蒸気エンジン): 1基あたり最低2048su/16rpm、条件次第で増大
